Enhancement Request: AI-Powered Dynamic Assessment Platform
Objective
Upgrade the existing assessment platform by introducing AI-generated questions, enhanced user engagement through rich micro-interactions, and a scalable architecture that supports future question types.

AI Question Generation
Requirements
Instead of storing all questions statically in the database, integrate Google Gemini API to dynamically generate questions based on:
Skill Category
Difficulty Level
Topic
Experience Level
Assessment Type
Learning Objectives
Current Scope
For Version 1, generate only Multiple Choice Questions (MCQs).
Each generated question must contain:
{
  "question": "Question text",
  "options": [
    "Option A",
    "Option B",
    "Option C",
    "Option D"
  ],
  "correctAnswer": "Option B",
  "explanation": "Detailed explanation",
  "difficulty": "Easy | Medium | Hard",
  "topic": "React Hooks"
}

Gemini Prompt Engineering
When generating questions:
Questions must be technically accurate.
Avoid duplicate questions.
Generate realistic industry-level scenarios.
Include explanations for every answer.
Ensure distractor options are meaningful.
Match the requested difficulty level.
Return structured JSON only.
Question Generation Flow
User starts assessment.
System displays loading state.
Gemini generates questions.
Questions are cached locally.
Assessment begins.
Results are evaluated instantly.
